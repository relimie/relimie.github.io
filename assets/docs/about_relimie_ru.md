## Для тех, кто хочет пить осознанно

Relimie — это не приложение для трезвости. Это осознанный помощник в отношениях с алкоголем — созданный для тех, кто любит выпить в компании, но хочет оставаться в курсе своих привычек без вины и осуждения.

В основе Relimie — твой **личный ориентир**: долгосрочная точка отсчёта, а не жёсткий ежедневный лимит. Воспринимай его как месячное среднее, вокруг которого ты стремишься держаться. Одни вечера выпьешь чуть больше, другие — меньше или совсем ничего, и это нормально. Главное — чтобы со временем твоё потребление возвращалось к ориентиру.

Твой светящийся **Орб** показывает положение дел с первого взгляда:

- **Зелёный** — ты уверенно в рамках ориентира
- **Золотой** — ты приближаешься к лимиту
- **Красный** — день восстановления поможет вернуть среднее в норму

Ориентир калибруется по **рекомендациям ВОЗ**, с учётом страны и пола. По мере улучшения привычек ты можешь его ужесточить — твой потолок должен становиться строже по мере того, как ты становишься сильнее.
